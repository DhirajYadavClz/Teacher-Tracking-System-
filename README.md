# Teacher-Tracking-System-
This project detects a teacher using face recognition and YOLO tracking,
and automatically calculates teaching time using a camera.

## Features
- Face recognition
- Person tracking with YOLOv8
- Teaching time calculation
- CSV logging

## Requirements
- Python 3.9+
- OpenCV
- face_recognition
- ultralytics

NOte :- Add your own teacher.jpg (Image of character to match face)
